目标导入模板说明

请遵循以下规范准备您的数据，以确保批量导入功能正常运行

模板结构规范

主目标行（Parent Goal）

  • 目标名称: 必填项，用于识别主目标。
  • 截止日期: 格式必须为 yyyy-MM-dd，如 2023-12-31。
  • 优先级: 仅接受 高、中、低。

子目标行（Sub-Goal）

  • 识别规则: 目标名称列“留空”即被识别为下方归属于最近主目标的子目标。
  • 子目标描述: 必填，用于描述具体拆解的任务。
  • 关联逻辑: 导入程序会自动将子目标挂载到其上方距离最近的有效主目标下。

快速校对清单

  1. 首行必须是标题行，数据从第二行开始录入。
  2. 日期列请统一设置为“文本格式”或标准的日期格式，避免导入乱码。
  3. 标签（Tags）支持多个，请使用英文逗号分隔，例如：健身,周计划,核心。
  4. 确保文件后缀为 .xlsx 或 .xls。
