# FreeMix 功能增强计划

本文档记录了 FreeMix 系统后续需要实现的三个核心功能增强点。旨在完善系统的"执行"与"回顾"能力。

## 1. 全局日历视图 (Global Calendar View) [优先实现]
**现状**: 系统目前仅支持列表和看板视图，缺乏基于时间维度的任务概览。
**目标**: 实现一个全屏日历视图，直观展示所有目标的截止日期和分布情况。
**功能细节**:
- **月视图/周视图切换**: 支持按月或按周查看任务。
- **任务展示**: 在日历格中显示当天截止的目标（带颜色区分优先级）。
- **拖拽排期**: 允许用户拖拽目标到其他日期，自动更新截止时间。
- **快速创建**: 点击日历空白处可快速创建该日期的目标。
- **筛选**: 支持按状态（未完成/已完成）和优先级筛选显示。

## 2. 数据导出与备份 (Data Export & Backup)
**现状**: 系统仅支持导入，缺乏导出功能，数据流动性差。
**目标**: 允许用户将目标数据导出为通用格式，保障数据主权。
**功能细节**:
- **多种格式**: 支持导出为 Excel (.xlsx) 和 Markdown (.md)。
- **选择性导出**: 支持导出全部数据或仅导出当前视图/筛选后的数据。
- **打印优化**: 提供适合打印的 PDF 导出样式（用于纸质复盘）。
- **自动备份**: (可选) 桌面端支持定时自动备份数据库到本地文件。

## 3. 目标富文本笔记 (Rich Text Notes for Goals)
**现状**: 目标仅有简单的文本描述，无法承载复杂的思考过程和资料。
**目标**: 为每个目标提供一个功能完善的富文本编辑器区域。
**功能细节**:
- **Markdown 支持**: 支持标题、列表、引用、代码块等标准 Markdown 语法。
- **多媒体嵌入**: 支持粘贴图片、插入链接。
- **实时保存**: 编辑内容自动保存，防止丢失。
- **关联引用**: (进阶) 支持引用其他目标或系统内的资源。
