快速上手指南

简单三步，开启您的目标管理之旅

01

下载安装

根据您的操作系统下载对应的安装包，按照提示完成安装。FreeMix 支持 macOS 和 Windows 系统，安装过程简单快捷。

02

创建目标

打开应用，点击"+"号创建您的第一个目标。您可以设置截止日期、优先级，并添加详细描述，让目标清晰可见。

03

多端同步

在任意设备上登录您的账户，您的数据将自动实时同步。无论在办公室还是在路上，随时随地掌控全局。
